# About Me

Hi, I’m Erkan Aksoy.

I’m a Senior Content Engineer at Hack The Box, where I create cybersecurity training content, hands-on labs, and realistic attack scenarios. My main areas of focus are Windows, Active Directory, Azure, and Microsoft Entra ID security.

Before moving into offensive security, I spent around ten years working as a systems administrator. That background still shapes how I approach security today: I like understanding how systems are designed, how they are managed in real environments, and how small configuration decisions can create unexpected attack paths.

My certifications include:

- **OffSec:** OSCP, OSWP, OSEP, OSED, OSWE (OSCE3)
- **Zero-Point Security:** CRTO (Certified Red Team Operator)
- **Altered Security:** CARTP (Certified Azure Red Team Professional), CARTE (Certified Azure Red Team Expert)

Certifications have been useful milestones, but most of my learning comes from building labs, researching attack techniques, breaking things, and documenting what I find.

## What I Write About

This blog mainly covers:

- Active Directory and identity security
- Azure and Microsoft Entra ID pentesting
- Windows and Linux security tooling
- Lab development and vulnerable machine design
- Technical research, experiments, and lessons learned
- Practical notes that may help other security professionals

My goal is to write content that is technically accurate, practical, and based on real testing rather than theory alone.
